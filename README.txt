# CS-340-Databases

Final Project:
A database driven website using HTML, PHP and MySQL.
The project "theme" is the NFL - Users can add/delete coaches, teams, players, Metros (cities), and positions.
Most are open-ended submissions.
On the team addition form, the second drop-down box populates from the currently existing Metro Table 

Live website link:
http://web.engr.oregonstate.edu/~mathewt/NFL/Test.php

Files:
All PDFs: Explanation of project, schema, diagrams
Remaining files are source code (including table creation SQL)
Applicable queries can be found in "NFLqueries.sql"
